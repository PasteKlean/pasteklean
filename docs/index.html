Installation

Safe to install

PasteKlean is not yet signed with an expensive OS certificate, so you may see a one-time security prompt. This is normal for indie software, and the app never collects any data.

Windows

Run the .exe installer. If Windows Defender or SmartScreen warns you, click More info then Run anyway.

macOS

Open the .dmg and drag PasteKlean into Applications. If Gatekeeper says the app cannot be opened, right-click the app icon and choose Open.
